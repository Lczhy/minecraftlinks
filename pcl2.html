PCL2 启动器

简介

PCL2（Plain Craft Launcher 2）是一款清爽、简洁的 Minecraft 启动器，由龙腾猫跃开发。它具有以下特点：

  • 界面美观，操作简单
  • 支持版本管理
  • 支持模组管理
  • 支持整合包制作
  • 内置游戏优化功能

使用教程

1. 下载并安装启动器

2. 首次运行时，选择 Java 路径

3. 登录你的 Minecraft 账号

4. 选择要启动的版本

5. 点击启动游戏

常见问题

如果遇到问题，可以：

  • 检查 Java 版本是否正确
  • 确认网络连接正常
  • 查看启动器日志
  • 访问官方帮助文档
